Bienvenue dans le SHELL interactif

Ce SHELL est compose de deux modes (le mode normal et le mode avec l'interface graphique) comprenants les memes options:
-Informations generales
-Reseaux
-Processus

L'option Information generales permet de recuperer des informations sur le systeme d'exploitation.

On peut obtenir dans cette option la version du systeme d'exploitation, l'uptime (le temps depuis lequel une  machine ou un logiciel tourne sans interuption), la version du kernel, les informations a propos du disque dur, la limite de fichiers et de processus ouverts et la liste des Paquets qui ont ete installe.


L'option Reseaux permet d'obtenir les informations sur le systeme reseaux.

On peut obtenir l'adresse IP de la connexion Ethernet, les differentes interfaces presentent dans le systeme d'exploitation, le nombre de Paquets transmis et le nombre de Paquets re√ßu,
le route permet d'afficher la table de routage, l'etat forward de Paquets permet de determiner si le routage internet est operationnel, le ping permet obtenir  le temps de reponse et de reception a travers un reseaux IP.
Pour finir les options activer et desactiver l'inferfaces Ethernet ou l'IP forward permettent d'obtenir les informations dans celles-ci sont activee.


L'option Processus permet d'obtenir des informations sur les processus et de les executer ou de les stopper.

On peut afficher tous les processus ce qui permet a l'utilisateur d'obtenir la liste de tous les processus present dans le systeme d'exploitation, afficher les details d'un processus ce qui  permet a l'utilisateurs d'obtenir les informations relative a ce processus et de l'arreter.
Pour finir l'option lancer le processus permet l'execution du processus entre en argument par l'utilisateur.
